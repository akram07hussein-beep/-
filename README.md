# إدارة العقارات والإيجارات — حزمة جاهزة للنشر

هذه حزمة Vite + React كاملة لتطبيقك. تبنيها مرة واحدة ثم ترفع مجلد `dist` على Netlify، وتثبّته على جوالك كتطبيق.

البيانات تُحفظ على جهازك (localStorage) داخل المتصفح/التطبيق، وتبقى محفوظة بين الجلسات. اعمل **نسخة احتياطية** دورياً من داخل التطبيق (المزيد ← نسخ احتياطي).

---

## المسار الأول: من حاسوب (الأوصى)

تحتاج تثبيت **Node.js** مرة واحدة من https://nodejs.org (نسخة LTS).

افتح موجّه الأوامر (Terminal / CMD) داخل مجلد المشروع ونفّذ:

```
npm install
npm run build
```

سيُنشأ مجلد **dist**. هذا هو التطبيق الجاهز.

للتجربة محلياً قبل النشر (اختياري):
```
npm run preview
```

### النشر على Netlify (سحب وإفلات)
1. ادخل https://app.netlify.com وسجّل الدخول.
2. اختر **Add new site ← Deploy manually**.
3. اسحب مجلد **dist** كاملاً وأفلته في الصفحة.
4. سيعطيك Netlify رابطاً مثل `your-app.netlify.app`. (يمكنك ربط نطاق مخصّص لاحقاً عبر Libyan Spider كما فعلت سابقاً.)

### النشر عبر GitHub (تحديثات تلقائية — اختياري)
ارفع المجلد كله إلى مستودع GitHub، ثم في Netlify اختر **Import from Git** وحدّد المستودع. الإعدادات مضبوطة في `netlify.toml` تلقائياً (build: `npm run build`، publish: `dist`).

---

## المسار الثاني: بدون حاسوب (من الجوال)

1. أنشئ حساباً على https://github.com من متصفح الجوال.
2. أنشئ مستودعاً جديداً وارفع ملفات هذه الحزمة (زر Add file ← Upload files — يمكنك رفع المجلدات).
3. في https://app.netlify.com اختر **Import from Git** وحدّد المستودع. سيبني Netlify التطبيق وينشره تلقائياً.

> ملاحظة: السحب والإفلات لمجلد `dist` يحتاج بناءً مسبقاً، لذا من الجوال استخدم مسار GitHub أعلاه (يبني Netlify نيابةً عنك).

---

## تثبيت التطبيق على الجوال (PWA)

بعد فتح الرابط على جوالك:

- **أندرويد (Chrome):** القائمة (⋮) ← **«إضافة إلى الشاشة الرئيسية»** / Install app.
- **آيفون (Safari):** زر المشاركة ← **«إضافة إلى الشاشة الرئيسية»**.

سيظهر التطبيق بأيقونته الخاصة، يعمل بكامل الشاشة، ودون إنترنت بعد أول فتح.

## بنية الملفات

```
property-app/
├─ index.html            ← الصفحة + Tailwind + بديل التخزين + تسجيل PWA
├─ package.json          ← الحزم وأوامر البناء
├─ vite.config.js
├─ netlify.toml
├─ src/
│  ├─ main.jsx           ← نقطة الدخول
│  └─ App.jsx            ← تطبيقك (المكوّن الكامل)
└─ public/
   ├─ manifest.webmanifest
   ├─ sw.js              ← عامل الخدمة (العمل دون إنترنت)
   └─ icon-192/512/180, favicon-32 (الأيقونات)
```

## تحديث التطبيق لاحقاً
عند أي تعديل جديد على الكود، استبدل محتوى `src/App.jsx` بالنسخة الجديدة، ثم أعد `npm run build` وارفع `dist` من جديد (أو ادفع التغيير إلى GitHub فيُحدّث تلقائياً).
